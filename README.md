OnlyLZ2
=======

楼主跟我走2

#关于软件
楼主跟我走是一个方便各位看论坛直播的使用者方便下载论坛直播贴全部脱水内容的软件

软件使用Java语言制作

#编译
您可以通过Git下载整个项目，之后通过IntelliJ IDEA直接打开项目

#运行
楼主跟我走使用Java制作，所以运行本软件必须要先安装Java环境

相关内容可以前往:

http://java.com

查看详细
